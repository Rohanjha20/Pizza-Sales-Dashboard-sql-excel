# Pizza Sales Dashboard (SQL Server + Excel)

## Project Overview
This project analyzes pizza sales data using SQL Server and presents business insights through an interactive Excel dashboard. It demonstrates an end-to-end data analytics workflow from raw data to decision-ready visualizations.

## Tools Used
- SQL Server Management Studio (SSMS)
- Microsoft Excel (Pivot Tables, Charts, Slicers, Timeline)
- SQL

## Key Metrics
- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Pizzas per Order

## Key Insights
- Highest orders occur from Thursday to Saturday evenings
- Peak order times are 12–1 PM and 4–8 PM
- Classic pizzas generate the highest sales and revenue
- Large-size pizzas contribute the most to total sales
- Pepperoni and Chicken pizzas are top sellers
- Brie Carre pizza is the lowest-performing item

## Project Workflow
1. Analyze raw pizza sales data using SQL queries
2. Export aggregated results to Excel
3. Build Pivot Tables and charts
4. Design an interactive dashboard with slicers and timeline

## How to Use
1. Run SQL queries in SQL Server
2. Export query results to Excel
3. Open the Excel dashboard file
4. Refresh Pivot Tables to explore insights
