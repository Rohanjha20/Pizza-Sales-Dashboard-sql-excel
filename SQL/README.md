# SQL Queries

This folder contains all SQL queries used in the Pizza Sales Dashboard project.

## KPIs
- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Pizzas per Order

## Analysis Performed
- Daily and hourly order trends
- Sales by pizza category and size
- Best and worst selling pizzas
- Monthly and quarterly filters using date functions

## Database
- SQL Server
- Table used: pizza_sales
